# Contributing to HOBrummen Pretband

Thanks for your interest in contributing! This project is the website for H.O.B. — the fun brass band from Brummen. Whether you're fixing a typo, improving the design, or adding a new feature, all contributions are welcome.

---

## Getting Started

1. **Fork** this repository and clone your fork locally:
   ```bash
   git clone https://github.com/your-username/pretband.git
   cd pretband
   ```

2. **Install dependencies** (if applicable):
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:3000` (or whatever port is configured).

---

## Making Changes

- Create a new branch for your changes:
  ```bash
  git checkout -b feature/your-feature-name
  ```
- Keep your changes focused — one feature or fix per branch/PR.
- Write clear, readable code and add comments where needed.

---

## Submitting a Pull Request

1. Push your branch to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
2. Open a **Pull Request** against the `main` branch of this repo.
3. Fill in a short description of what you changed and why.
4. A maintainer will review your PR and get back to you.

---

## Reporting Issues

Found a bug or have a suggestion? Open an [issue](https://github.com/HOBrummen/pretband/issues) and include:
- A clear title and description
- Steps to reproduce (for bugs)
- Screenshots if relevant

---

## Guidelines

- Be respectful and friendly — this is a community project for a local band 🎺
- Keep the tone of the website fun and in the spirit of the band
- Don't commit directly to `main`
- Test your changes before opening a PR

---

## Questions?

Feel free to open an issue or reach out to the maintainers. We're happy to help!
